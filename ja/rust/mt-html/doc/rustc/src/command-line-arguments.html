Command-line argumentsコマンドライン引数


Here's a list of command-line arguments to rustc and what they do. rustcコマンドライン引数のリストと、それらが何をしているのかを以下に示します。


-h / --help : get help -h / --help ：ヘルプを表示する


This flag will print out help information for rustc .このフラグはrustcヘルプ情報をrustcます。


--cfg : configure the compilation environment --cfg ：コンパイル環境を設定する


This flag can turn on or off various #[cfg] settings.このフラグは、さまざまな#[cfg]設定をオンまたはオフにすることができます。


-L : add a directory to the library search path -L ：ライブラリ検索パスにディレクトリを追加する


When looking for external crates, a directory passed to this flag will be searched.外部の箱を探すとき、このフラグに渡されたディレクトリが検索されます。


-l : link the generated crate to a native library -l ：生成されたクレートをネイティブライブラリにリンクします。


This flag allows you to specify linking to a specific native library when building a crate.このフラグを使用すると、クレートを作成するときに特定のネイティブライブラリへのリンクを指定できます。


--crate-type : a list of types of crates for the compiler to emit --crate-type ：コンパイラが--crate-typeクレートのタイプのリスト


This instructs rustc on which crate type to build.これは、 rustc木箱のタイプを作るかをrustcに指示します。


--crate-name : specify the name of the crate being built --crate-name ：ビルドされるクレートの名前を指定する


This informs rustc of the name of your crate.これはあなたの木箱の名前をrustcに伝えます。


--emit : emit output other than a crate --emit ：クレート以外の出力を出力する


Instead of producing a crate, this flag can print out things like the assembly or LLVM-IR.このフラグは、クレートを生成する代わりに、アセンブリやLLVM-IRのようなものを印刷することができます。


--print : print compiler information --print ：コンパイラ情報を出力する


This flag prints out various information about the compiler.このフラグは、コンパイラに関するさまざまな情報を表示します。


-g : include debug information -g ：デバッグ情報を含む


A synonym for -C debug-level=2 . -C debug-level=2同義語です。


-O : optimize your code -O ：コードを最適化する


A synonym for -C opt-level=2 . -C opt-level=2同義語。


-o : filename of the output -o ：出力のファイル名


This flag controls the output filename.このフラグは、出力ファイル名を制御します。


--out-dir : directory to write the output in --out-dir ：出力を書き込むディレクトリ


The outputted crate will be written to this directory.出力された箱はこのディレクトリに書き込まれます。


--explain : provide a detailed explanation of an error message --explain ：エラーメッセージの詳細な説明を提供する


Each error of rustc 's comes with an error code; rustcの各エラーにはエラーコードが付いています。 this will print out a longer explanation of a given error.これは与えられたエラーのより長い説明を表示します。


--test : build a test harness --testテスト：テストハーネスを構築する


When compiling this crate, rustc will ignore your main function and instead produce a test harness.この箱をコンパイルするとき、 rustcはあなたのmain機能を無視し、代わりにテストハーネスを生成します。


--target : select a target triple to build --target ：構築するためにトリプルターゲットを選択


This controls which target to produce.これは、生成するtargetを制御します。


-W : set lint warnings -W ：lint警告を設定する


This flag will set which lints should be set to the warn level .このフラグは、どのlintを警告レベルに設定するかを設定します 。


-A : set lint allowed -A ：lintの設定を許可する


This flag will set which lints should be set to the allow level .このフラグは、どのlintを許可レベルに設定するかを設定します 。


-D : set lint denied -D ：lintが拒否されました。


This flag will set which lints should be set to the deny level .このフラグは、どのlintを拒否レベルに設定するかを設定します 。


-F : set lint forbidden -F ：lintの禁止を設定する


This flag will set which lints should be set to the forbid level .このフラグは、どのlintを禁止レベルに設定するかを設定します 。


--cap-lints : set the most restrictive lint level --cap-lints ：最も限定的なlintレベルを設定する


This flag lets you 'cap' lints, for more, see here .このフラグは、lintsを 'キャップする'ことができます。詳しくは、 こちらをご覧ください 。


-C / --codegen : code generation options -C / --codegen ：コード生成オプション


This flag will allow you to set codegen options .このフラグを使用すると、 codegenオプションを設定できます 。


-V / --version : print a version -V / --version ：バージョンを出力する


This flag will print out rustc 's version.このフラグはrustcのバージョンを出力します。


-v / --verbose : use verbose output -v / --verbose ：冗長出力を使用する


This flag, when combined with other flags, makes them produce extra output.このフラグは、他のフラグと組み合わされると、余分な出力を生成します。


--extern : specify where an external library is located --extern ：外部ライブラリの場所を指定する


This flag allows you to pass the name and location of an external crate that will be linked into the crate you're buildling.このフラグはビルドしているクレートにリンクされる外部クレートの名前と場所を渡すことができます。


--sysroot : Override the system root --sysroot ：システムルートを上書きする


The "sysroot" is where rustc looks for the crates that come with the Rust distribution; "sysroot"はrustcがRustディストリビューションに付属する箱を探す場所です。 this flag allows that to be overridden.このフラグは、このフラグによって上書きされます。


--error-format : control how errors are produced --error-format ：エラーの発生方法を制御する


This flag lets you control the format of errors.このフラグを使用すると、エラーのフォーマットを制御できます。


--color : configure coloring of output --color ：出力の色付けを設定する


This flag lets you control color settings of the output.このフラグを使用すると、出力のカラー設定を制御できます。
