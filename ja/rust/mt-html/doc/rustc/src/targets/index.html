Targetsターゲット


rustc is a cross-compiler by default. rustcはデフォルトでクロスコンパイラです。 This means that you can use any compiler to build for any architecture.つまり、任意のコンパイラを使用して任意のアーキテクチャ用にビルドすることができます。 The list of targets are the possible architectures that you can build for. ターゲットのリストは、構築可能なアーキテクチャーです。


To see all the options that you can set with a target, see the docs here .ターゲットで設定できるすべてのオプションを表示するには、こちらのドキュメントを参照してhere 。
