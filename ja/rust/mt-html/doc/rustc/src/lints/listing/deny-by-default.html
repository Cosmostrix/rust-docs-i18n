legacy-directory-ownershipレガシーディレクトリ所有権


The legacy_directory_ownership warning is issued when legacy_directory_ownership警告は、


There is a non-inline module with a # path attribute (eg # path = "foo.rs" mod bar;), ＃ path属性を持つ非インラインモジュールがあります（例： path = "foo.rs" mod bar;）
The module's file ("foo.rs" in the above example) is not named "mod.rs", andモジュールのファイル（上記の例では "foo.rs"）は "mod.rs"という名前ではありません。
The module's file contains a non-inline child module without a # path attribute.モジュールのファイルに、＃ path属性のないインラインでない子モジュールが含まれています。

The warning can be fixed by renaming the parent module to "mod.rs" and moving it into its own directory if appropriate.警告は、親モジュールを "mod.rs"に変更し、必要に応じてそれを独自のディレクトリに移動することで修正できます。
