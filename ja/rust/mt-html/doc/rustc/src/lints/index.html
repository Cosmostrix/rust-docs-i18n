Lints糸くず


In software, a "lint" is a tool used to help improve your source code.ソフトウェアでは、「lint」はソースコードの改善に役立つツールです。 The Rust compiler contains a number of lints, and when it compiles your code, it will also run the lints. Rustコンパイラには多数のlintが含まれており、コードをコンパイルするときにlintも実行されます。 These lints may produce a warning, an error, or nothing at all, depending on how you've configured things.これらの糸くずは、どのように設定したかによって、警告、エラー、まったく何も表示されません。
