What is rustc? rustcとは何ですか？


Welcome to "The rustc book"! "錆びた本"へようこそ！ rustc is the compiler for the Rust programming language, provided by the project itself. rustcは、プロジェクト自身が提供するRustプログラミング言語のコンパイラです。 Compilers take your source code and produce binary code, either as a library or executable.コンパイラは、ソースコードを取得し、バイナリコードをライブラリまたは実行可能ファイルとして生成します。


Most Rust programmers don't invoke rustc directly, but instead do it through Cargo .ほとんどのRustプログラマーはrustc直接呼び出すのではなく、代わりにCargoます。 It's all in service of rustc though!しかし、それはrustcたサービスのすべてです！ If you want to see how Cargo calls rustc , you can Cargoがrustcと呼ぶ方法を見たい場合は、
