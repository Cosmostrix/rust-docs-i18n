This feature allows the generation of code coverage reports.この機能により、コードカバレッジレポートの生成が可能になります。


Set the -Zprofile compiler flag in order to enable gcov profiling. gcovプロファイリングを有効にするには、 -Zprofileコンパイラフラグを設定します。


For example: ` Bash cargo new testgcov --bin cd testgcov export RUSTFLAGS="-Zprofile" cargo build cargo run `例： ` Bash cargo new testgcov --bin cd testgcov export RUSTFLAGS="-Zprofile" cargo build cargo run `


Once you've built and run your program, files with the gcno (after build) and gcda (after execution) extensions will be created.プログラムをビルドして実行すると、 gcno （ビルド後）とgcda （実行後）という拡張子を持つファイルが作成されます。 You can parse them with llvm-cov gcov or grcov .あなたはllvm- grcov gcovまたはgrcovそれらを解析できます。
