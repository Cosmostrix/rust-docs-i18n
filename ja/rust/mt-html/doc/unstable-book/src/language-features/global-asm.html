The global_asm€ macro allows the programmer to write arbitrary assembly outside the scope of a function body, passing it through rustc and llvm to the assembler. global_asm€マクロを使用すると、プログラマーは関数本体のスコープの外側に任意のアセンブリを書き込んで、 rustcとllvmを介してrustcに渡すことができます。 The macro is a no-frills interface to LLVM's concept of module-level inline assembly .このマクロは、LLVMのmodule-level inline assemblyのコンセプトに対するノンフリルインターフェイスです。 That is, all caveats applicable to LLVM's module-level inline assembly apply to global_asm€ .つまり、LLVMのモジュールレベルのインラインアセンブリに適用できるすべての警告がglobal_asm€適用されます。

global_asm€ fills a role not currently satisfied by either asm€ or #[naked] functions. global_asm€は、現在asm€関数または#[naked]関数では満たされていないロールを埋めます。 The programmer has all features of the assembler at their disposal.プログラマは、アセンブラのすべての機能を自由に使用できます。 The linker will expect to resolve any symbols defined in the inline assembly, modulo any symbols marked as external.リンカーは、インラインアセンブリで定義されたシンボルを解決し、外部としてマークされたシンボルをモジュロすることを期待します。 It also means syntax for directives and assembly follow the conventions of the assembler in your toolchain.また、ディレクティブとアセンブリの構文は、ツールチェーン内のアセンブラの規則に従っています。
