The extern_in_paths feature allows to refer to names from other crates "inline", without introducing extern crate items, using keyword extern . extern_in_paths機能はextern crate項目を導入せずにキーワードexternを使って他の箱の名前を "インライン"で参照することを可能にします。


For example, extern::my_crat::a::b will resolve to path a::b in crate my_crate .たとえば、 extern::my_crat::a::bパスに解決されますa::bクレートでmy_crate 。


feature(extern_absolute_paths) mode provides the same effect by resolving absolute paths like ::my_crate::a::b to paths from extern crates by default. feature(extern_absolute_paths)モードは、 ::my_crate::a::bような絶対パスをデフォルトでexternクレートのパスに解決することによって同じ効果をもたらします。


Suppose we have a dependency crate xcrate available through Cargo.toml , or --extern options, or standard Rust distribution, or some other means. Cargo.toml 、-- --externオプション、標準的なRustディストリビューション、またはその他の手段で利用可能な依存関係クレートxcrateます。
