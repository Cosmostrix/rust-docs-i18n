The MSVC ABI on x86 Windows uses the thiscall calling convention for C++ instance methods by default; x86 Windows上のMSVC ABIは、デフォルトでC ++インスタンスメソッドに対してthiscall呼び出し規約を使用します。 it is identical to the usual (C) calling convention on x86 Windows except that the first parameter of the method, the this pointer, is passed in the ECX register.これは、メソッドの最初のパラメータ（ thisポインタ）がECXレジスタに渡される点を除いて、x86 Windowsの通常の（C）呼び出し規約と同じです。
