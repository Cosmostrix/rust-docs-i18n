In the MSP430 architecture, interrupt handlers have a special calling convention. MSP430アーキテクチャでは、割り込みハンドラに特別な呼び出し規約があります。 You can use the "msp430-interrupt" ABI to make the compiler apply the right calling convention to the interrupt handlers you define. "msp430-interrupt" ABIを使用して、コンパイラが正しい呼び出し規約を定義した割り込みハンドラに適用させることができます。
