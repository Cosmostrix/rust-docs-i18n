Tool lints let you use scoped lints, to allow , warn , deny or forbid lints of certain tools.ツールリントは、特定のツールのリントをallow 、 warn 、 denyまたはforbid warn 、スコープ付きリントを使用します。


Currently clippy is the only available lint tool.現時点でclippyが使用可能な唯一のリントツールです。


It is recommended for lint tools to implement the scoped lints like this:糸くずツールでは、以下のように範囲指定されたlintを実装することをお勧めします。


#[_(TOOL_NAME::lintname)] : for lint names #[_(TOOL_NAME::lintname)] ：リント名の場合
#[_(TOOL_NAME::lintgroup)] : for groups of lints #[_(TOOL_NAME::lintgroup)] ：lintsのグループ用
#[_(TOOL_NAME::all)] : for (almost ^1 ) all lints #[_(TOOL_NAME::all)] ：すべてのlintを（ほぼ^1 ）
