At present, literals are only accepted as the value of a key-value pair in attributes.現在、リテラルは属性内のキーと値のペアの値としてのみ受け入れられます。 What's more, only string literals are accepted.さらに、 文字列リテラルのみが受け入れられます。 This means that literals can only appear in forms of #[attr(name = "value")] or #[attr = "value"] .これは、リテラルが#[attr(name = "value")]または#[attr = "value"]形式でしか表示されないことを意味します。
