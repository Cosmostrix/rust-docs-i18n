The unboxed_closures feature allows you to write functions using the "rust-call" ABI, required for implementing the `Fn*` family of traits. unboxed_closures機能を使うと、 `Fn*`ファミリの特性を実装するために必要な"rust-call" ABIを使って関数を書くことができます。 "rust-call" functions must have exactly one (non self) argument, a tuple representing the argument list. "rust-call"関数は、引数リストを表すタプルという、ちょうど1つ（非自己）の引数を持たなければなりません。
