The crate_in_paths feature allows to explicitly refer to the crate root in absolute paths using keyword crate . crate_in_paths特徴は、明示的にキーワードの使用絶対パスでクレートルートを参照することを可能にするcrate 。


crate can be used only in absolute paths, ie either in ::crate::a::b::c form or in use items where the starting :: is added implicitly. crateのいずれかで、すなわち、 唯一絶対パスで使用することができる::crate::a::b::c形又はにおけるuse開始がアイテム::暗黙的に追加されます。 Paths like crate::a::b::c are not accepted currently. crate::a::b::cようなパスは現在受け入れられていません。


This feature is required in feature(extern_absolute_paths) mode to refer to any absolute path in the local crate (absolute paths refer to extern crates by default in that mode), but can be used without feature(extern_absolute_paths) as well.この機能は、 feature(extern_absolute_paths)モードではローカルクレートの絶対パスを参照する必要があります（絶対パスはそのモードではデフォルトでexternクレートを参照します）が、 feature(extern_absolute_paths)なしでも使用できます。
