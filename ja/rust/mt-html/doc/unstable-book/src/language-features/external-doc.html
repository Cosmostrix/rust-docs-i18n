The external_doc feature allows the use of the include parameter to the #[doc] attribute, to include external files in documentation. external_doc機能は、 #[doc]に外部ファイルを含めるために、 #[doc]属性へのincludeパラメータの使用を許可します。 Use the attribute in place of, or in addition to, regular doc comments and #[doc] attributes, and rustdoc will load the given file when it renders documentation for your crate.通常の文書コメントと#[doc]属性の代わりに、またはそれに加えて属性を使用すると、 rustdocはあなたの木枠の文書をレンダリングするときに指定のファイルを読み込みます。


With the following files in the same directory:同じディレクトリに次のファイルがあるとします。


external-doc.md : external-doc.md ：



lib.rs : lib.rs ：



rustdoc will load the file external-doc.md and use it as the documentation for the MyAwesomeType struct. rustdocはexternal-doc.mdというファイルを読み込み、 MyAwesomeType構造体のドキュメントとして使用します。


When locating files, rustdoc will base paths in the src/ directory, as if they were alongside the lib.rs for your crate.ファイルを見つけるとき、 rustdocはsrc/ディレクトリ内のパスを、あなたのクレート用のlib.rsします。 So if you want a docs/ folder to live alongside the src/ directory, start your paths with ../docs/ for rustdoc to properly find the file.したがって、 docs/フォルダをsrc/ディレクトリの../docs/てrustdocたい場合は、 rustdoc docs/で../docs/パスを見つけてファイルを見つけてください。
