The doc_cfg feature allows an API be documented as only available in some specific platforms. doc_cfg機能を使用すると、特定のプラットフォームでのみ使用可能なAPIとしてドキュメント化できます。 This attribute has two effects:この属性には2つの効果があります。


In the annotated item's documentation, there will be a message saying "This is supported on (platform) only".注釈付きアイテムのドキュメントには、「これは（プラットフォーム）のみでサポートされています」というメッセージが表示されます。

The item's doc-tests will only run on the specific platform.アイテムのdoc-testsは、特定のプラットフォームでのみ実行されます。

This feature was introduced as part of PR #43348 to allow the platform-specific parts of the standard library be documented.この機能は、標準ライブラリのプラットフォーム固有の部分を文書化できるように、PR #43348一部として導入されました。


The application's icon in the notification area (aka system tray).通知領域（システムトレイとも呼ばれます）にあるアプリケーションのアイコン。
