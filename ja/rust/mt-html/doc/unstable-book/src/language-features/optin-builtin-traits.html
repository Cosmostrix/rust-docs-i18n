The optin_builtin_traits feature gate allows you to define auto traits. optin_builtin_traits機能ゲートを使用すると、自動特性を定義できます。


Auto traits, like `Send` or `Sync` in the standard library, are marker traits that are automatically implemented for every type, unless the type, or a type it contains, has explicitly opted out via a negative impl.標準ライブラリの`Send`や`Sync`ような自動形質は、型や型に含まれていない限り、負のimplで明示的にオプトアウトしていない限り、すべての型に対して自動的に実装されるマーカーの特性です。
