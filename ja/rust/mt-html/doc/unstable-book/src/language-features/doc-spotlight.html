The doc_spotlight feature allows the use of the spotlight parameter to the #[doc] attribute, to "spotlight" a specific trait on the return values of functions. doc_spotlight機能では、 #[doc]属性にspotlightパラメータを使用して、関数の戻り値の特定の特性を「スポットライト」することができます。 Adding a #[doc(spotlight)] attribute to a trait definition will make rustdoc print extra information for functions which return a type that implements that trait.特性定義に#[doc(spotlight)]属性を追加すると、rustdocは、その特性を実装する型を返す関数の追加情報を表示します。 This attribute is applied to the Iterator , io::Read , and io::Write traits in the standard library.この属性は、標準ライブラリのIterator 、 io::Read 、およびio::Write特性に適用されます。
