Passesパス


Rustdoc has a concept called "passes". Rustdocには「パス」という概念があります。 These are transformations that rustdoc runs on your documentation before producing its final output.これは、最終出力を生成する前に、ドキュメント上でrustdocが実行する変換です。


In addition to the passes below, check out the docs for these flags:以下のパスに加えて、これらのフラグのドキュメントをチェックしてください：


--passes
--no-defaults

Default passesデフォルトパス


By default, rustdoc will run some passes, namely:デフォルトでは、rustdocはいくつかのパスを実行します。


strip-hidden
strip-private
collapse-docs
unindent-comments

However, strip-private implies strip-private-imports , and so effectively, all passes are run by default.しかし、 strip-private strip-private-imports意味します。したがって、すべてのパスはデフォルトで実行されます。


strip-hidden


This pass implements the #[doc(hidden)] attribute.このパスは#[doc(hidden)]属性を実装しています。 When this pass runs, it checks each item, and if it is annotated with this attribute, it removes it from rustdoc 's output.このパスが実行されると、各項目がチェックされ、この属性で注釈が付けられていれば、 rustdocの出力から削除されます。


Without this pass, these items will remain in the output.このパスがなければ、これらの項目は出力に残ります。


unindent-comments


When you write a doc comment like this:このような文書コメントを書くときは：


This is a documentation comment.これはドキュメントのコメントです。

There's a space between the /// and that T . ///とそのT間にはスペースがあります。 That spacing isn't intended to be a part of the output;その間隔は、出力の一部になるようには意図されていません。 it's there for humans, to help separate the doc comment syntax from the text of the comment.それは人間のために、コメントのテキストからドキュメンテーションのコメントの構文を分離するのに役立ちます。 This pass is what removes that space.このパスは、そのスペースを削除するものです。


The exact rules are left under-specified so that we can fix issues that we find.正確なルールは不足しているため、見つけた問題を修正できます。


Without this pass, the exact number of spaces is preserved.このパスがなければ、正確なスペース数は保持されます。


collapse-docs


With this pass, multiple #[doc] attributes are converted into one single documentation string.このパスでは、複数の#[doc]属性が1つのドキュメント文字列に変換されます。

strip-private


This removes documentation for any non-public items, so for example:これにより、公開されていないアイテムのドキュメントは削除されます。例：


These are private docs.これらはプライベートドキュメントです。
These are public docs.これらは公開ドキュメントです。

This pass removes the docs for Private , since they're not public.このパスは公開されていないため、 Private用のドキュメントを削除します。


This pass implies strip-priv-imports .このパスはstrip-priv-importsを意味strip-priv-importsます。


strip-priv-imports


This is the same as strip-private , but for extern crate and use statements instead of items.これはstrip-privateと同じですが、項目の代わりにextern crateとuse文のためです。
