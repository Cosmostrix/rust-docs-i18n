What is rustdoc? rustdocとは何ですか？


The standard Rust distribution ships with a tool called rustdoc .標準Rustディストリビューションにはrustdocというツールがrustdocます。 Its job is to generate documentation for Rust projects.その仕事は、Rustプロジェクトのドキュメントを生成することです。 On a fundamental level, Rustdoc takes as an argument either a crate root or a Markdown file, and produces HTML, CSS, and JavaScript.根本的なレベルでは、Rustdocは、クレートルートまたはMarkdownファイルを引数としてとり、HTML、CSS、およびJavaScriptを生成します。
