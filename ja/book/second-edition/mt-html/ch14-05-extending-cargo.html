Extending Cargo with Custom Commandsカスタムコマンドによる貨物の拡張


Cargo is designed so you can extend it with new subcommands without having to modify Cargo. Cargoは、貨物を変更することなく新しいサブコマンドで拡張できるように設計されています。 If a binary in your $PATH is named cargo-something , you can run it as if it was a Cargo subcommand by running cargo something .あなたの中のバイナリた場合$PATH命名されたcargo-something 、それは実行して、貨物のサブコマンドであるかのように、あなたはそれを実行することができcargo something 。 Custom commands like this are also listed when you run cargo --list . cargo --listを実行すると、このようなカスタムコマンドも表示されます。 Being able to use cargo install to install extensions and then run them just like the built-in Cargo tools is a super convenient benefit of Cargo's design!使用することができることcargo install拡張機能をインストールした後、同じようにそれらを実行する組み込みの貨物ツール貨物の設計の超便利な利点があります！
