Common Programming Concepts共通プログラミングの概念


This chapter covers concepts that appear in almost every programming language and how they work in Rust.この章では、ほぼすべてのプログラミング言語で表示される概念と、Rustでどのように動作するかについて説明します。 Many programming languages have much in common at their core.多くのプログラミング言語は、共通点が共通しています。 None of the concepts presented in this chapter are unique to Rust, but we'll discuss them in the context of Rust and explain the conventions around using these concepts.この章で紹介するコンセプトは、Rust独自のものではありませんが、Rustのコンテキストで説明し、これらのコンセプトを使用するための規則を説明します。


Specifically, you'll learn about variables, basic types, functions, comments, and control flow.具体的には、変数、基本タイプ、関数、コメント、および制御フローについて学習します。 These foundations will be in every Rust program, and learning them early will give you a strong core to start from.これらの基盤はすべてのRustプログラムに含まれています。早期に学習することで、まずは強力なコアとなります。


Keywordsキーワード


The Rust language has a set of keywords that are reserved for use by the language only, much as in other languages. Rust言語には、他の言語と同じように、その言語でのみ使用するために予約された一連のキーワードがあります。 Keep in mind that you cannot use these words as names of variables or functions.変数や関数の名前としてこれらの単語を使用することはできません。 Most of the keywords have special meanings, and you'll be using them to do various tasks in your Rust programs;ほとんどのキーワードは特別な意味を持っています。これらのキーワードを使用してRustプログラムでさまざまなタスクを実行します。 a few have no current functionality associated with them but have been reserved for functionality that might be added to Rust in the future.それらに関連する現在の機能はありませんが、今後はRustに追加される可能性のある機能のために予約されています。 You can find a list of the keywords in Appendix A.付録Aにキーワードのリストがあります。
