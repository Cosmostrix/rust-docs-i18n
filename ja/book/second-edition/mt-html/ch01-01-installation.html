Installationインストール


The first step is to install Rust.最初のステップは、Rustをインストールすることです。 We'll download Rust through rustup , a command line tool for managing Rust versions and associated tools. Rustバージョンと関連するツールを管理するためのコマンドラインツール、 rustupを通じてRustをダウンロードします。 You'll need an internet connection for the download.ダウンロードにはインターネット接続が必要です。


Note: If you prefer not to use rustup for some reason, please see the Rust installation page for other options.注：何らかの理由でrustupないようにしたい場合は、Rustのインストールページを参照してください。


The following steps install the latest stable version of the Rust compiler.以下の手順で最新の安定バージョンのRustコンパイラをインストールします。 All the examples and output in this book use stable Rust 1.21.0.本書のすべての例と出力では、安定したRust 1.21.0を使用しています。 Rust's stability guarantees ensure that all the examples in the book that compile will continue to compile with newer Rust versions. Rustの安定性は、コンパイルされた本のすべての例が新しいRustバージョンでコンパイルされ続けることを保証します。 The output might differ slightly between versions, because Rust often improves error messages and warnings. Rustはエラーメッセージと警告を改善することが多いため、出力はバージョンによって若干異なる場合があります。 In other words, any newer, stable version of Rust you install using these steps should work as expected with the content of this book.言い換えれば、これらの手順を使用してインストールする新しい、安定版のRustは、この本の内容で期待どおりに動作するはずです。


Command Line Notationコマンドライン表記法


In this chapter and throughout the book, we'll show some commands used in the terminal.この章と本では、端末で使用されるコマンドをいくつか示します。 Lines that you should enter in a terminal all start with $ .ターミナルに入力する行はすべて$始まります。 You don't need to type in the $ character; $文字を入力する必要はありません。 it indicates the start of each command.各コマンドの開始を示します。 Lines that don't start with $ typically show the output of the previous command. $始まらない行は通常、前のコマンドの出力を表示します。 Additionally, PowerShell-specific examples will use > rather than $ .さらに、PowerShell固有の例では、 $ではなく>が使用されます。

Local Documentationローカルドキュメント


The installer also includes a copy of the documentation locally, so you can read it offline.インストーラーには、ドキュメンテーションのコピーもローカルに含まれているため、オフラインで読むことができます。 Run rustup doc to open the local documentation in your browser. rustup docを実行して、ブラウザのローカルrustup docを開きます。


Any time a type or function is provided by the standard library and you're not sure what it does or how to use it, use the application programming interface (API) documentation to find out!標準ライブラリによって型または関数が提供され、その動作や使用方法がわからない場合は、アプリケーションプログラミングインターフェイス（API）のドキュメントを参照してください。
