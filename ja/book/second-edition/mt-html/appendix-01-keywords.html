Appendix A: Keywords付録A：キーワード


The following list contains keywords that are reserved for current or future use by the Rust language.以下のリストには、Rust言語による現在または将来の使用のために予約されているキーワードが含まれています。 As such, they cannot be used as identifiers, such as names of functions, variables, parameters, struct fields, modules, crates, constants, macros, static values, attributes, types, traits, or lifetimes.したがって、関数、変数、パラメータ、構造体フィールド、モジュール、クレート、定数、マクロ、静的値、属性、型、特性、または寿命の名前など、識別子として使用することはできません。


Keywords Currently in Use現在使用中のキーワード


The following keywords currently have the functionality described.次のキーワードは、現在説明されている機能を備えています。


as - perform primitive casting, disambiguate the specific trait containing an item, or rename items in use and extern crate statements - 、原始的な鋳造を行う項目を含む特定の形質を明確に、または内のアイテムの名前を変更しuseしてextern crateステートメント
break - exit a loop immediately break - すぐにループを終了する
const - define constant items or constant raw pointers const - 定数項目または定数生ポインタを定義する
continue - continue to the next loop iteration continue - 次のループの繰り返しに進む
crate - link an external crate or a macro variable representing the crate in which the macro is defined crate - マクロが定義されているクレートを表す外部クレートまたはマクロ変数をリンクする
else - fallback for if and if let control flow constructs else - ifおよびif letためのフォールバック
enum - define an enumeration enum - enum定義する
extern - link an external crate, function, or variable extern - 外部の枠、関数、または変数をリンクする
false - Boolean false literal false - ブール値falseリテラル
fn - define a function or the function pointer type fn - 関数または関数のポインタ型を定義する
for - loop over items from an iterator, implement a trait, or specify a higher-ranked lifetime for -iteratorの項目をループしたり、特性を実装したり、より高いランクのライフタイムを指定することができます
if - branch based on the result of a conditional expression if - 条件式の結果に基づく分岐
impl - implement inherent or trait functionality impl - 固有または特性機能を実装する
in - part of for loop syntax in - forループ構文の一部
let - bind a variable let - 変数をバインドする
loop - loop unconditionally loop -無条件ループ
match - match a value to patterns match - 値をパターンに一致させる
mod - define a module mod - モジュールを定義する
move - make a closure take ownership of all its captures move - クロージャがすべてのキャプチャの所有権を取得するようにする
mut - denote mutability in references, raw pointers, or pattern bindings mut - 参照、生ポインタ、またはパターンバインディングにおける変更可能性を示す
pub - denote public visibility in struct fields, impl blocks, or modules pub - 構造体フィールド、 implブロック、またはモジュールのパブリックな可視性を示します
ref - bind by reference ref参照
return - return from function return - 関数からの戻り値
Self - a type alias for the type implementing a trait Self - 特性を実装する型の型別名
self - method subject or current module self試験の主題または現在のモジュール
static - global variable or lifetime lasting the entire program execution static - プログラム実行全体を持続させるグローバル変数または生涯
struct - define a structure struct定義する
super - parent module of the current module現在のモジュールのsuper親モジュール
trait - define a trait trait - trait定義する
true - Boolean true literal true - ブール値trueリテラル
type - define a type alias or associated type type - 型のエイリアスまたは関連する型を定義する
unsafe - denote unsafe code, functions, traits, or implementations unsafe - 危険なコード、関数、特性、実装を示す
use - import symbols into scope use - シンボルをスコープにインポートする
where - denote clauses that constrain a type where - 型を制約する節を示す
while - loop conditionally based on the result of an expression while - 式の結果に基づいて条件付きでループする
