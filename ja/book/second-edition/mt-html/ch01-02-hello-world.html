If you're more familiar with a dynamic language, such as Ruby, Python, or JavaScript, you might not be used to compiling and running a program as separate steps. Ruby、Python、JavaScriptなどの動的言語に慣れている場合は、プログラムをコンパイルして別の手順として実行することに慣れていない可能性があります。 Rust is an ahead-of-time compiled language, meaning you can compile a program and give the executable to someone else, and they can run it even without having Rust installed. Rustは事前にコンパイルされた言語です。つまり、プログラムをコンパイルして実行可能ファイルを他の人に与えることができ、Rustをインストールしなくても実行できます。 If you give someone a .rb , .py , or .js file, they need to have a Ruby, Python, or JavaScript implementation installed (respectively).誰かに.rb 、 .py 、または.jsファイルを与える場合は、Ruby、Python、またはJavaScriptの実装をそれぞれインストールする必要があります。 But in those languages, you only need one command to compile and run your program.しかし、これらの言語では、プログラムをコンパイルして実行するためのコマンドが1つだけ必要です。 Everything is a trade-off in language design.すべてが言語デザインのトレードオフです。
