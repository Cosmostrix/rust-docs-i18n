Vectorsベクトル


A 'vector' is a dynamic or 'growable' array, implemented as the standard library type Vec<T> . 「ベクトル」は、標準的なライブラリタイプVec<T>として実装された動的または拡張可能な配列です。 The T means that we can have vectors of any type (see the chapter on generics for more). Vectors store their contents as contiguous arrays of T on the heap.ベクトルはその内容をヒープ上に連続したT配列として格納します。 This means that they must be able to know the size of T at compile time (that is, how many bytes are needed to store a T ?).つまり、コンパイル時にTのサイズ（つまり、 Tを格納するために必要なバイト数）を知ることができなければなりません。 The size of some things can't be known at compile time.コンパイル時にはいくつかのサイズを知ることはできません。 For these you'll have to store a pointer to that thing: thankfully, the Box type works perfectly for this.これらのためには、そのことへのポインタを格納する必要があります：ありがたいことに、 Box型はこれに対して完全に機能します。
