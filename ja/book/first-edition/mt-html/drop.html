So what is Drop good for?では、 Dropは何のために良いのですか？ Generally, Drop is used to clean up any resources associated with a struct .一般に、 Dropは、 struct関連付けられたリソースをクリーンアップするために使用されます。 For example, the Arc<T> type is a reference-counted type.たとえば、 Arc<T>型は参照カウント型です。 When Drop is called, it will decrement the reference count, and if the total number of references is zero, will clean up the underlying value. Dropが呼び出されると、参照カウントがデクリメントされ、参照の総数がゼロの場合、基になる値がクリーンアップされます。
