Syntax and Semantics構文とセマンティクス


This chapter breaks Rust down into small chunks, one for each concept.この章では、Rustを各概念ごとに1つずつ小さな塊に分割しています。


If you'd like to learn Rust from the bottom up, reading this in order is a great way to do that. Rustをボトムアップから学びたい場合は、これを順番に読むことが最適な方法です。


These sections also form a reference for each concept, so if you're reading another tutorial and find something confusing, you can find it explained somewhere in here.これらのセクションは各コンセプトのリファレンスとなりますので、別のチュートリアルを読んで紛らわしいものを見つけた場合は、ここで説明しています。
