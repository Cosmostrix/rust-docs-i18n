Iterator::any


Iterator::any is a function which when passed an iterator, will return true if any element satisfies the predicate. Otherwise false. Its signature:


The type being iterated over.
any takes &mut self meaning the caller may be borrowed and modified, but not consumed.
FnMut meaning any captured variable may at most be modified, not consumed. Self::Item states it takes arguments to the closure by value.

iter() for vecs yields &i32. Destructure to i32.
into_iter() for vecs yields i32. No destructuring required.
iter() for arrays yields &i32.
into_iter() for arrays unusually yields &i32.
