Designators


The arguments of a macro are prefixed by a dollar sign $ and type annotated with a designator:


This macro takes an argument of designator ident and creates a function named $func_name. The ident designator is used for variable/function names.
The stringify€ macro converts an ident into a string.
Create functions named foo and bar with the above macro.
This macro takes an expression of type expr and prints it as a string along with its result. The expr designator is used for expressions.
stringify€ will convert the expression as it is into a string.
Recall that blocks are expressions too!

This is a list of all the designators:


block
expr is used for expressions
ident is used for variable/function names
item
pat (pattern)
path
stmt (statement)
tt (token tree)
ty (type)
