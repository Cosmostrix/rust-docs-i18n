Iterator::anyイテレータ::任意


Iterator::any is a function which when passed an iterator, will return true if any element satisfies the predicate. Iterator::anyは、イテレータを渡すと、いずれかの要素が述語を満たす場合にtrueを返す関数です。 Otherwise false .それ以外の場合はfalse 。 Its signature:その署名：


The type being iterated over.オーバーレイされる型。
any takes &mut self meaning the caller may be borrowed and modified, but not consumed. anyテイク&mut selfは、呼び出し元を借りて変更することができますが、消費しないことを意味します。
FnMut meaning any captured variable may at most be modified, not consumed. FnMutは、キャプチャされた変数が最大でも変更され、消費されないことを意味します。 Self::Item states it takes arguments to the closure by value. Self::Itemは、値によってクロージャへの引数をとります。

iter() for vecs yields &i32 . vecのiter()は&i32生成します。 Destructure to i32 . Destruct to i32 。
into_iter() for vecs yields i32 . into_iter()はi32生成します。 No destructuring required.破壊は必要ありません。
iter() for arrays yields &i32 .配列のiter()は&i32生成します。
into_iter() for arrays unusually yields &i32 .配列のinto_iter()は&i32異常に生成します。
