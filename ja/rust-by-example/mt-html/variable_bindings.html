Variable Bindings変数バインディング


Rust provides type safety via static typing. Rustは静的タイピングを介して型の安全性を提供します。 Variable bindings can be type annotated when declared.変数バインディングには、宣言されたときに注釈を付けることができます。 However, in most cases, the compiler will be able to infer the type of the variable from the context, heavily reducing the annotation burden.しかし、ほとんどの場合、コンパイラはコンテキストの変数の型を推測し、注釈の負担を大幅に減らすことができます。


Values (like literals) can be bound to variables, using the let binding.値（リテラルなど）は、 letバインディングを使用して変数にバインドできます。
