Of course trait s can also be generic.もちろん、 trait一般的でもあり得る。 Here we define one which reimplements the Drop trait as a generic method to drop itself and an input.ここでは、 Drop traitを、それ自体と入力をdropする一般的な方法として再実装するものを定義します。


Non-copyable types.コピー不可タイプ。
A trait generic over T . T以上の特性。
Define a method on the caller type which takes an additional single parameter T and does nothing with it.追加の単一パラメータTを取り、それには何もしない呼び出し元の型のメソッドを定義します。
Implement DoubleDrop<T> for any generic parameter T and caller U .任意の汎用パラメータTと呼び出し元Uに対してDoubleDrop<T>を実装します。
This method takes ownership of both passed arguments, deallocating both.このメソッドは、渡された両方の引数の所有権を取得し、両方を解放します。
