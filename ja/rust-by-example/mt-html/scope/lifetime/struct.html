Annotation of lifetimes in structures are also similar to functions:構造内の生存時間の注釈も関数に似ています。


A type Borrowed which houses a reference to an i32 . i32への参照を格納するタイプBorrowed 。 The reference to i32 must outlive Borrowed . i32への参照はBorrowedなければなりません。
Similarly, both references here must outlive this structure.同様に、ここでの両方の参照は、この構造より長く存続しなければならない。
An enum which is either an i32 or a reference to one. i32または1への参照のいずれかである列挙型。
