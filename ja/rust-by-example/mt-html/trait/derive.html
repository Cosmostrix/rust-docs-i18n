The compiler is capable of providing basic implementations for some traits via the #[derive] attribute .コンパイラは、 #[derive] attribute介していくつかの特性の基本的な実装を提供することができattribute 。 These traits can still be manually implemented if a more complex behavior is required.より複雑な振る舞いが必要な場合、これらの特性は手動で実装することができます。
