Result implements FromIter so that a vector of results ( Vec<Result<T, E>> ) can be turned into a result with a vector ( Result<Vec<T>, E> ). Resultは、結果のベクトル（ Vec<Result<T, E>> ）をベクトル（ Result<Vec<T>, E> ）で結果に変換できるようにFromIter実装します。 Once an Result::Err is found, the iteration will terminate. Result::Errが見つかると、繰り返しは終了します。
