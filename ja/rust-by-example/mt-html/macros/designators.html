Designators指定子


The arguments of a macro are prefixed by a dollar sign $ and type annotated with a designator :マクロの引数にはドル記号$と接頭辞が付いていて、 指定子で注釈が付けられます。


This macro takes an argument of designator ident and creates a function named $func_name .このマクロは、指定子ident引数をとり、 $func_nameという名前の関数を作成します。 The ident designator is used for variable/function names. ident指定子は、変数/関数名に使用されます。
The stringify€ macro converts an ident into a string. stringify€マクロは、 identを文字列に変換します。
Create functions named foo and bar with the above macro.上記のマクロを使ってfooとbarという名前の関数を作成します。
This macro takes an expression of type expr and prints it as a string along with its result.このマクロはexpr型の式をとり、その結果とともに文字列として出力します。 The expr designator is used for expressions. expr指定子は式に使用されます。
stringify€ will convert the expression as it is into a string. stringify€はそのまま式を文字列に変換します。
Recall that blocks are expressions too!ブロックも式であることを思い出してください！
