The use declaration use宣言


The use declaration can be used to bind a full path to a new name, for easier access. use宣言を使用して、フルパスを新しい名前にバインドして、より簡単にアクセスすることができます。


Bind the deeply::nested::function path to other_function . other_function deeply::nested::functionパスをother_functionます。

This is equivalent to use deeply::nested::function as function .これはuse deeply::nested::function as function 。 This function() will shadow the outer one.このfunction()は外側のものをシャドウします。
use bindings have a local scope. useバインディングにはローカルスコープがあります。 In this case, the shadowing of function() is only in this block.この場合、 function()シャドーイングはこのブロック内にのみあります。
