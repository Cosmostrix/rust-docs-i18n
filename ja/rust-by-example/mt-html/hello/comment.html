Commentsコメント


Any program requires comments and indeed Rust supports a few different varieties:どんなプログラムでもコメントが必要です.Rustはいくつかの異なる種類をサポートしています。


Regular comments which are ignored by the compiler:コンパイラによって無視される通常のコメント ：
// Line comments which go to the end of the line.
/* Block comments which go to the closing delimiter. */

println!("Hello, world!"); println！（ "こんにちは、世界！"）;
Run it.それを実行します。 See?見る？ Now try deleting the two slashes, and run it again. 2つのスラッシュを削除して、もう一度実行してみましょう。
Observe how block comments allow easy expression manipulation which line comments do not.ブロックコメントが、どの行コメントでも簡単に式を操作できるようにする。 Deleting the comment delimiters will change the result:コメント区切り文字を削除すると、結果が変更されます。
