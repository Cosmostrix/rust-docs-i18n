In general, the {} will be automatically replaced with any arguments.一般的に、 {}は自動的に引数に置き換えられます。 These will be stringified.これらはストリング化されます。
Without a suffix, 31 becomes an i32.接尾辞がなければ、31はi32になります。 You can change what type 31 is, with a suffix.タイプ31は、接尾辞付きで変更できます。
There are various optional patterns this works with.これにはさまざまなオプションパターンがあります。 Positional arguments can be used.位置的な引数を使うことができます。
As can named arguments.名前付き引数と同じです。
Special formatting can be specified after a : .特別な書式設定は以下の後に指定することができます: 。
You can right-align text with a specified width.テキストを指定した幅で右揃えにすることができます。 This will output " 1".これは "1"を出力します。 5 white spaces and a "1". 5の空白と「1」。
You can pad numbers with extra zeroes.あなたは余分なゼロでパッドすることができます。 This will output "000001".これで "000001"が出力されます。
It will even check to make sure the correct number of arguments are used.正しい数の引数が使用されているかどうかをチェックすることさえできます。
