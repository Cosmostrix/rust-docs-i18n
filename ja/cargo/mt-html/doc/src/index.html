The Cargo Bookカーゴブック


! ！ Cargo Logo カーゴロゴ


Cargo is the Rust package manager . CargoはRust パッケージマネージャーです。 Cargo downloads your Rust project's dependencies, compiles your project, makes packages, and upload them to crates.io , the Rust community's package registry . CargoはRustプロジェクトの依存関係をダウンロードし、プロジェクトをコンパイルし、パッケージを作成し、Rustコミュニティのパッケージレジストリである crates.ioにアップロードします 。 You can contribute to this book on GitHub .あなたはGitHubこの本に貢献することができGitHub 。
