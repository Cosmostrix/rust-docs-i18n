Build cacheキャッシュを構築する


Cargo shares build artifacts among all the packages of a single workspace.貨物のシェアは、単一のワークスペースのすべてのパッケージ間にアーティファクトを構築します。 Today, Cargo does not share build results across different workspaces, but a similar result can be achieved by using a third party tool, sccache .今日では、Cargoは異なるワークスペース間でビルド結果を共有していませんが、サードパーティのツールであるsccacheを使用しても同様の結果が得られます。


To setup sccache , install it with cargo install sccache and set RUSTC_WRAPPER environmental variable to sccache before invoking Cargo. sccacheを設定するには、 cargo install sccacheを使用してsccacheをインストールし、Cargoを呼び出す前にRUSTC_WRAPPER環境変数をsccacheに設定します。 If you use bash, it makes sense to add export RUSTC_WRAPPER=sccache to .bashrc . bashを使用している場合は、 export RUSTC_WRAPPER=sccacheを.bashrcに追加することが理にexport RUSTC_WRAPPER=sccacheてい.bashrc 。 Refer to sccache documentation for more details.詳細については、sccacheのマニュアルを参照してください。
