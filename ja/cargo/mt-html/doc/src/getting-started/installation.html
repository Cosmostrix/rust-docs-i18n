Install Rust and Cargo錆と貨物をインストールする


The easiest way to get Cargo is to install the current stable release of Rust by using rustup .カーゴを入手する最も簡単な方法は、現在の安定したリリースのRustをrustupてインストールすることrustup 。


On Linux and macOS systems, this is done as follows: LinuxおよびmacOSシステムでは、これは次のように行われます。



It will download a script, and start the installation.スクリプトをダウンロードしてインストールを開始します。 If everything goes well, you'll see this appear:すべてがうまくいくと、これが表示されます：



On Windows, download and run rustup-init.exe . Windowsでは、 rustup-init.exeダウンロードして実行します。 It will start the installation in a console and present the above message on success.コンソールにインストールを開始し、上記のメッセージを表示します。


After this, you can use the rustup command to also install beta or nightly channels for Rust and Cargo.その後、 rustupコマンドを使用して、RustとCargoのbetaチャンネルまたはnightlyチャンネルをインストールすることもできます。


For other installation options and information, visit the install page of the Rust website.その他のインストールオプションと情報については、Rust Webサイトのinstallページを参照してください。

Alternatively, you can build Cargo from source .また、 ソースから貨物を構築することもできます。
