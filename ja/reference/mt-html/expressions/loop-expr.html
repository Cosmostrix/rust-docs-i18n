Rust supports four loop expressions: Rustは4つのループ式をサポートしています：


A loop expression denotes an infinite loop. loop式は、無限ループを表します。
A while expression loops until a predicate is false. while式は述語がfalseになるまでループします。
A while let expression tests a refutable pattern. while let式は不変のパターンをテストします。
A for expression extracts values from an iterator, looping until the iterator is empty. for式はイテレータから値を抽出し、イテレータが空になるまでループします。

All four types of loop support break expressions , continue expressions , and labels . 4つのタイプのループはすべて、 式の break式の continue 、およびlabelsサポートします。 Only loop supports evaluation to non-trivial values . loopのみが評価を非自明な値にサポートします 。
