This example generates thumbnails for all .jpg files in the current directory then saves them in a new folder called thumbnails .この例では、現在のディレクトリにあるすべての.jpgファイルのサムネイルを作成し、 thumbnailsと呼ばれる新しいフォルダに保存します。
