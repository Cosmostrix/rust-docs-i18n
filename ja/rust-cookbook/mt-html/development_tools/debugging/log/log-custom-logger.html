Implements a custom logger ConsoleLogger which prints to stdout. stdoutに出力するカスタムロガーConsoleLoggerを実装します。 In order to use the logging macros, ConsoleLogger implements the `log::Log` trait and `log::set_logger` installs it.ロギングマクロを使用するために、 ConsoleLoggerは`log::Log`特性を実装し、 `log::set_logger`はそれをインストールします。
