Recursively calculate file sizes at given depth与えられた深さでファイルサイズを再帰的に計算する


! ！ walkdir-badge ! walkdir-badge ！ cat-filesystem-badge


Recursion depth can be flexibly set by `WalkDir::min_depth` & `WalkDir::max_depth` methods.再帰深度は、 `WalkDir::min_depth`と`WalkDir::max_depth`メソッドによって柔軟に設定できます。 Calculates sum of all file sizes to 3 subfolders depth, ignoring files in the root folder.すべてのファイルサイズの合計を3つのサブフォルダの深さに計算し、ルートフォルダ内のファイルは無視します。
