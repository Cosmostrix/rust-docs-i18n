Find loops for a given path指定されたパスのループを見つける


! ！ same_file-badge ! same_file-badge ！ cat-filesystem-badge


Use `same_file::is_same_file` to detect loops for a given path. `same_file::is_same_file`を使用して、指定されたパスのループを検出します。 For example, a loop could be created on a Unix system via symlinks: ` bash mkdir -p /tmp/foo/bar/baz ln -s /tmp/foo/ /tmp/foo/bar/baz/qux ` The following would assert that a loop exists.たとえば、シンボリックリンクを介してUnixシステム上にループを作成することができます： ` bash mkdir -p /tmp/foo/bar/baz ln -s /tmp/foo/ /tmp/foo/bar/baz/quxループが存在すると主張する。
